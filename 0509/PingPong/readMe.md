프로젝트 개요
PingPong은 캔버스 위에 겨울 풍경을 그리고 애니메이션으로 눈이 내리는 안드로이드 앱입니다. 사용자는 눈의 크기와 개수를 실시간으로 조절할 수 있습니다.
기능

✨ 실시간 눈 내리는 애니메이션
🎚️ 눈 크기 조절 (5~35 범위)
📊 눈 개수 조절 (10~200개)
⏯️ 애니메이션 시작/일시정지
🎨 아름다운 겨울 풍경 배경
🔄 부드러운 20 FPS 애니메이션
